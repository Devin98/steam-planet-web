# 一笼星球前端项目

**这是一个音乐爱好者的音乐推送App**
  
后端部分[点这里](https://github.com/Devin98/steam-planet-server)

## 前端技术栈

**主要就是vue全家桶...组件库用了有赞开源的vant**

1. vue2
1. vuex
1. vue-router
1. axios
1. vant

## 开发环境

**确保node环境且版本≥8**

1. `git clone git@github.com:Devin98/steam-planet-web.git`
1. `cd steam-plant-web`
1. `yarn`
1. `yarn dev`
